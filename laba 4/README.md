## Задача 2.2 Поиск максимума
```
Создайте метод, принимающий набор Коробок из задачи 4.1.1 и возвращающий максимальное из их значений в формате double. 
Принимаемые методом Коробки могут быть параметризованы любыми видами чисел.
```
В основной программе создаются 3 коробки, которые являются объектами класса **Box** типов: **int**,**double**,**float**.  
Затем последовательно заполняем коробки с клавиатуры, проверяя каждый элемент через метод **Metods.isNumber()**. Далее эти коробки 
передаём в массив коробок, который может принимать коробки содержащие свойство любого типа, являющиеся **Number** или наследующиеся от **Number**.
Соответственно передаём этот массив в метод **maxValue(boxes)**, где находим максимальное значение и выводим его.  
Для решения в классе **Main** был создан метод **double maxValue(Box<? extends Number>[] boxes)** для нахождения максимального значения
```java
if (boxes == null || boxes.length == 0) {
    throw new IllegalArgumentException("Массив коробок не может быть пустым или null");
}

boolean found = false;//флаг, если мы нашли хоть одно значение в коробке
double max = 0.0;// значение максимума
for (Box<? extends Number> s : boxes) { // проходимся по массиву коробок
    if (s != null && !s.isEmpty()) { //если коробка не пуста и не null
        double value = s.getObject().doubleValue(); //берём значение из коробки
        if (!found) { //если фаг false
            max = value;// приравниваем значение из коробки к максимуму
            found = true; //флаг true, что хотябы 1 коробка не пуста
        } else if (value > max) {
            max = value;
        }
    }
}
if (!found) { //если все коробки пусты или null
    throw new IllegalStateException("Не найдено ни одной непустой коробки");
}
return max; //возвращаем максимум
```
В решении используется класс **Box<T>**, который описан в задаче 1.1

```
Создадим 2 коробку.Введите значение типа double: -2

```
